- HP: 12 | Defense: 5 (leather armor)

**Outcome:**
- If Urg escapes to cavern: He's alive, ritual incomplete, goblins rally
- If Urg defeated: Goblins lose compulsion, many flee, Razgut fights to death
- If Razgut defeated: Goblins demoralized, easier pursuit

**OPTION C: TACTICAL NEGOTIATION (Risky)**

*Difficulty:* Extreme (Deception DC 7 or Intimidate DC 7)

**Setup:** Party reveals presence, demands negotiation

**Razgut's Response:** *"You? One human? You defeated Scar Ridge? Perhaps... you're more dangerous than we thought. What do you want, little hero?"*

**Negotiation Paths:**

1. **Demand Release** - Razgut laughs. "We serve *the Deep Father*. He demands war. Negotiations are for the weak."

2. **Exploit Goblin Resentment** - Can party convince goblins they're *enslaved*? (Very hard)
   - If successful: Goblin uprising, goblins turn on Urg and Razgut
   - If failed: Combat advantage lost, surrounded

3. **Offer Alliance** - "Join us against the Deep Father." (Nearly impossible)
   - Razgut sees this as weakness
   - Only option if party seems unstoppable (rare)

4. **Demand Urg** - "Give us the Shaman, we leave." (Possible compromise)
   - Razgut considers: Urg is valuable, but not irreplaceable
   - If desperate, might offer Urg to buy time
   - Goblins would fight to keep caves secret

**Outcome:** Negotiation only delays combat. Used well, gains intelligence. Used poorly, worse tactical position.
